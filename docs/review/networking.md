# 🧠 Преговор: NETWORKING

<div class="flashcard"><div class="question">Коя команда в Linux показва кои портове са "слушащи" (Listening)?</div><div class="answer"><code>netstat -tlnp</code></div><div class="details"><b>-t</b>: TCP портове<br>
        <b>-l</b>: Listening (слушащи)<br>
        <b>-n</b>: Цифрови адреси (по-бързо)<br>
        <b>-p</b>: Показва кой процес (PID) ползва порта.</div></div>
